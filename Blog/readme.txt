Welcome to Blogueuese. 

Flask development framework. The site uses the Jinja2 templating engine is fully server-rendered with no accompanying front-end framework. Data is stored in a SQLite database and is accessed via direct SQL queries instead of an ORM. Input sanitization is achived via Bleach and WTForms was used for validation.

Stock images form Unsplash. Html templates are based on free templates from HTML5Up

See app.py for the entry point.
